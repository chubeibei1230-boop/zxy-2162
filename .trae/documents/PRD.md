## 1. 产品概述

培训资料包分装管理工具是一款纯前端的轻量级管理系统，用于培训课程资料包的分装内容记录、批次编号管理和缺漏确认。系统支持三类角色协同工作，所有数据草稿保存在浏览器 localStorage 中，无需后端服务即可独立运行。

- **核心价值**：规范培训资料分装流程，减少人工差错，提升复核效率
- **目标用户**：培训机构教务管理者、资料分装执行者、质量复核人员

---

## 2. 核心功能

### 2.1 用户角色

| 角色 | 进入方式 | 核心权限 |
|------|----------|----------|
| 管理者 | 角色切换 | 维护课程批次、管理资料模板、查看全部记录、生成发放单 |
| 执行者 | 角色切换 | 填写实际分装数量、新增分装记录、查看负责的任务 |
| 复核者 | 角色切换 | 确认缺漏情况、填写补装说明、批量设置复核状态 |

### 2.2 功能模块

1. **角色切换区**：顶部快速切换当前操作角色，界面随角色动态调整可操作项
2. **资料记录列表**：核心工作区，展示所有分装记录，支持增删改查
3. **筛选过滤栏**：多维度筛选记录集合
4. **批次管理**：课程批次的新增、编辑、删除
5. **资料模板管理**：预设资料模板，一键生成分装条目
6. **发放单预览**：只读模式下的发放清单，按课程批次分组展示
7. **自动校验提示**：实时数据校验，异常项高亮提醒

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 主工作台 | 角色切换栏 | 顶部标签切换管理者/执行者/复核者三种视角 |
| 主工作台 | 统计概览卡 | 总记录数、缺漏数、已复核比例等关键指标 |
| 主工作台 | 筛选工具栏 | 课程、资料类别、复核状态、负责人员、是否缺漏筛选 |
| 主工作台 | 记录列表 | 表格形式展示分装记录，支持行内编辑、按批次折叠 |
| 主工作台 | 批量操作栏 | 批量设置复核状态、批量删除 |
| 批次管理弹窗 | 课程批次列表 | 维护课程名称与批次号的对应关系 |
| 模板管理弹窗 | 资料模板列表 | 维护资料名、类别、包内数量等模板数据 |
| 发放单预览 | 分组清单 | 按课程批次分组的只读发放清单，含统计汇总 |

## 3. 核心流程

### 3.1 典型工作流

管理者先创建课程批次和资料模板 → 执行者根据模板生成分装条目并填写实际数量 → 复核者检查并确认缺漏情况、填写补装说明 → 管理者生成发放单预览。

### 3.2 流程图

```mermaid
flowchart LR
    A["管理者创建课程批次"] --> B["管理者维护资料模板"]
    B --> C["执行者从模板生成分装条目"]
    C --> D["执行者填写实际准备数量"]
    D --> E["系统自动校验数据"]
    E --> F{校验通过?}
    F -- 否 --> G["高亮提示异常项"]
    F -- 是 --> H["复核者进行复核"]
    H --> I["填写缺漏说明与补装备注"]
    I --> J["设置复核状态"]
    J --> K["管理者生成发放单预览"]
```

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：深海蓝 `#1e3a5f`，代表专业与可信赖
- **辅助色**：琥珀橙 `#f59e0b`（缺漏警示）、翡翠绿 `#10b981`（已通过）、玫瑰红 `#f43f5e`（错误）
- **背景色**：浅灰蓝渐变 `#f1f5f9 → #e2e8f0`，清爽不刺眼
- **按钮风格**：微圆角（6px），悬停时有轻微上浮与阴影加深
- **字体**：标题使用思源黑体 / Noto Sans SC 粗体，正文使用系统无衬线字体
- **布局风格**：左侧侧边栏 + 右侧内容区的经典管理后台布局，卡片式信息容器
- **图标风格**：简约线性图标，统一 16px 尺寸

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI 元素 |
|----------|----------|---------|
| 主工作台 | 顶部导航 | 系统标题、角色切换标签、发放单预览入口 |
| 主工作台 | 统计卡片区 | 4 张数据卡片横向排列，带数字与趋势 |
| 主工作台 | 筛选栏 | 下拉选择器 + 搜索框 + 折叠/展开按钮 |
| 主工作台 | 记录表格 | 斑马纹行、悬停高亮、异常行红色边框、批次分组行 |
| 主工作台 | 操作列 | 编辑、删除、复核状态切换等行内操作 |
| 弹窗 | 表单对话框 | 半透明遮罩 + 居中白色卡片 + 底部操作按钮 |
| 发放单预览 | 分组清单 | 按批次分组的卡片列表，打印友好样式 |

### 4.3 响应式

- 桌面端（≥1280px）：侧边栏 + 内容区双栏布局，表格完整展示所有列
- 平板端（768px-1279px）：侧边栏可折叠为图标模式，表格横向滚动
- 移动端（<768px）：顶部导航堆叠，统计卡片垂直排列，表格改为卡片列表展示

## 5. 数据校验规则

| 校验项 | 规则描述 | 异常提示方式 |
|--------|----------|--------------|
| 批次号重复 | 同一课程下批次号不可重复 | 红色边框 + 工具提示 |
| 数量不足 | 实际准备数量 < 包内数量 | 行高亮 + 警示图标 |
| 缺漏说明为空 | 有缺漏但缺漏说明为空 | 红色文字提醒 |
| 状态不一致 | 复核状态为"通过"但存在缺漏 | 状态标签闪烁提醒 |
| 任务过载 | 同一负责人负责超过 8 条记录 | 负责人名称橙色高亮 |
